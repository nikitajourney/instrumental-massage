# Инструментальный массаж - онлайн-курс

Лендинг онлайн-курса по инструментальному массажу.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Для GitHub Pages используется статическая сборка:

```bash
npm run build:pages
```

Важно: GitHub Pages публикует только статический фронтенд. Серверный обработчик `/api/lead` для отправки заявок требует отдельного Node.js-хостинга.

## Заявки на GitHub Pages

На GitHub Pages форма использует статический email-обработчик FormSubmit и отправляет заявки на `academia.massage@yandex.ru`.

При первой заявке FormSubmit может прислать письмо для подтверждения адреса получателя. После подтверждения следующие заявки будут приходить на почту автоматически.

Если нужен собственный backend, задайте `VITE_LEAD_API_URL` или запускайте Node.js-сервер с `/api/lead`.

## Telegram-уведомления

Telegram-уведомления работают только через backend. Не добавляйте `TELEGRAM_BOT_TOKEN` в `VITE_*` переменные и не публикуйте токен в GitHub Pages.

Для backend-хостинга задайте переменные окружения:

```bash
TELEGRAM_BOT_TOKEN="..."
TELEGRAM_CHAT_ID="..."
```

Пользователь или группа, куда должны приходить заявки, должны сначала написать боту `/start`. После этого `chat_id` можно получить через метод Telegram Bot API `getUpdates`.
